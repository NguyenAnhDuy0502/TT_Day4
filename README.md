# Landing Page Project

Đây là một **landing page** được thiết kế đẹp mắt với nhiều hiệu ứng mượt mà, viết bằng HTML, CSS và JavaScript.

## 🔧 Công nghệ sử dụng
- HTML
- CSS3 + Flexbox / Grid
- JavaScript thuần (Vanilla JS)
- Hiệu ứng CSS và animation (transitions, hover, scroll effects)

## 🚀 Cách chạy dự án

1. Mở **VS Code** và mở thư mục chứa dự án của bạn.
2. Mở tệp `index.html`.
3. Click chuột phải vào `index.html` → chọn **Open with Live Server**.
4. Trình duyệt sẽ tự động mở trang landing page với đầy đủ hiệu ứng.
